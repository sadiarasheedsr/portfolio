# Sadia Rasheed Portfolio Website

A modern personal portfolio website built with **HTML5**, **CSS3**, and **JavaScript**.  
It presents personal information, skills, education, achievements, contact links, and a downloadable CV in a smooth, responsive layout.

## Features

- Clean and responsive single-page portfolio design
- Hero section with intro, buttons, and profile image
- About Me section with personal details
- Animated skills progress bars
- Education timeline/cards
- Achievements section with certification highlights
- Contact section with Gmail, LinkedIn, and GitHub links
- Smooth scrolling and hover animations
- Glassmorphism / Pinterest-inspired visual style

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
